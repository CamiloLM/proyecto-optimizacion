# Optimización 2026-1 — Caso de Curso

## Descripción

Este repositorio contiene el desarrollo del caso de optimización del curso, en el cual se analizan dos problemas clásicos de Optimización. Para cada problema se incluye la formulación matemática, la solución obtenida mediante herramientas computacionales (principalmente Python y Gurobi), el análisis de los resultados y una revisión de literatura sobre su importancia y aplicaciones.

Adicionalmente, se desarrolla un modelo de redes complementario, correspondiente al análisis de la red de interacciones de personajes de la serie *Juego de Tronos*.
